== Installation ==

1. Create directory `/wp-content/plugins/monetization-with-blarchivescom`
1. Upload `blarchives.php` to the `/wp-content/plugins/monetization-with-blarchivescom` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a publisher account at https://blarchives.com/publishers/signup/
1. Obtain your blog ID by clicking the "Details" button next to your blog.
1. Under Settings - Blarchives, enter your Blarchives.com username and blog id.
1. Customize any other settings.
1. Make posts preium: either set dates in the settings page, or manually make
posts premium with the Blarchives control on every post's edit page.

== Upgrade Notice ==

All settings are carried over automatically.

Automatic upgrade: just click update to install the new version.  

Manual upgrade: download the zip file and unzip.  Replace the file
`wp-content/plugins/monetization-with-blarchivescom/blarchives.php` with the
new `blarchives.php` from the zip file.
